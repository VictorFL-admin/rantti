# 📝 Instrucciones para usar Sanity CMS

## 🚀 Inicio Rápido

### 1. Acceder al Studio
Abre tu navegador y ve a:
```
http://localhost:3000/studio
```

O ejecuta en la terminal:
```bash
npm run sanity
```
Esto abrirá el Studio en: `http://localhost:3333`

### 2. Iniciar sesión
- Haz clic en "Sign in"
- Usa tu cuenta de Sanity (la que creamos con tu email)

### 3. Crear el primer documento
1. En el menú lateral, verás "Página de Inicio"
2. Haz clic en el botón "+" o "Create"
3. Verás todos los campos para editar

## 📋 Campos editables

### 🎯 Sección Principal
- **Etiqueta Superior**: El badge que aparece arriba
- **Título Principal**: El título grande de la página
- **Descripción**: El texto debajo del título
- **Botón Principal**: Texto del botón principal
- **Botón Secundario**: Texto del botón secundario

### 📋 Cómo Funciona
- **Badge**: "Proceso Simple"
- **Título**: "¿Cómo funciona?"
- **Título Resaltado**: La parte en azul
- **Subtítulo**: Texto descriptivo
- **Pasos para Vender**: 4 pasos (cada uno con título y descripción)
- **Pasos para Comprar**: 4 pasos (cada uno con título y descripción)
- **Textos de botones y CTA final**

### ⭐ Beneficios
- **Badge, Título, Subtítulo**
- **Lista de Beneficios**: Puedes agregar hasta 6 beneficios

### 💼 Servicios
- **Información del servicio destacado**
- **Lista de beneficios**

### 🚀 Llamado Final
- **Todos los textos del CTA final**
- **3 características** con íconos
- **Texto del botón y texto de confianza**

## ✏️ Cómo editar

1. Haz clic en el campo que quieres cambiar
2. Escribe el nuevo texto
3. Haz clic en "Publish" (botón verde en la parte superior derecha)
4. ¡Listo! Los cambios aparecerán en tu sitio en ~1 minuto

## 🆘 Soporte

Si tienes problemas:
1. Revisa que estés logueado en Sanity
2. Verifica que hayas publicado los cambios
3. Espera 1-2 minutos para que se actualice el sitio
4. Recarga la página del sitio

## ⚙️ Comandos útiles

```bash
# Iniciar el Studio de Sanity
npm run sanity

# Desplegar el Studio a la nube
npm run sanity:deploy
```

Después de desplegar, podrás acceder al Studio desde cualquier lugar en:
```
https://rantti.sanity.studio
```
